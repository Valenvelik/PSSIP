При разработке данного сайта использовались теги:

ТЕКСТ:
<h1> - заголовки для текста
	<h1>Винсмок Санджи</h1>
<p> - абзац
	<p>Абзац</p>
<br> - перенос на новую строку
	</table><br>
<b> - жирный
	<
<i> - курсив
<u> - подчеркнутый
<s> - зачеркнутый
<sup> - верхний индекс
<sub> - нижний индекс
<font> - изменять цвета

СПИСКИ:
<ol> - нумерованный список
<ul> - маркерованный список

ТАБЛИЦЫ:
<table> - таблица
<tbody> - тело таблицы
<tr> - строки
<td> - столбцы
colspan - объединение колонок(столбцов)
rowspan - объединение строк
border - граница

ГИПЕРССЫЛКИ:
<a> - ссылка
<a href = #название> - якорная ссылка

МУЛЬТИМЕДИА:
<img> - картинки
<video> - видеофайл
<audio> - аудиофайлы

ФОРМЫ:
<form> - задаем форму
<radio> - переключатели
<checkbox> - чек бокс
<label> - надпись
<select> - выбрать
<legeng> - незвние элемента формы
<option> - выпадающий список
placeholder - отображение текста внутри label
